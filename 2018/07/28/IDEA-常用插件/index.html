IDEA常用插件

Translation

翻译插件，支持谷歌翻译、百度翻译、有道翻译，看源码注释的一大利器

String Manipulation

主要功能：变量驼峰式转换，其他功能未关注

soapUI Plugin

soap接口调试插件，如WebService接口调试

Scala

开发Scala程序必备插件

Restfultoolkit

扫描工作空间下所有项目中的rest接口，即所有的Controller

Python

开发Python必备插件

NodeJS

NodeJS相关插件，比如可以直接在package.json中执行相关命令启动项目，或一键编译运行node项目

###MyBatisCodeHelperPro

mybatis插件，支持mapper.java与mapper.xml映射，方便查找方法对应的sql内容等

Mongo Plugin

MongoDB插件，与Idea的Datasource功能类似，配置MongoDB服务，正常操作MongoDB数据库

Maven Helper

maven插件，可查看jar包依赖关系，自定义maven命令

Lombok Plugin

lombok插件，对lombok框架的idea支持

JSON Viewer

json格式化工具

Jenkins Control Plugin

jenkins控制

Ledis

redis客户端

GsonFormat

json格式化，可将json转换成POJO类

GradleDependencyHelper

gradle插件，可查看gradle依赖，不过没有maven helper方便

Gradle/Maven Navigation

gradle和maven依赖互转，可将maven依赖复制到gradle配置文件中，会自动转换为gradle的依赖配置。

Go

开发Go的必备插件

CheckStyle-IDEA

checkstyle插件，默认提供sun和google的代码风格

Alibaba Java Coding Guidelines

阿里巴巴代码检查插件，会按照阿里巴巴的编程规范进行代码检查

actiBPM

activity工作流流程图界面，不过感觉和其他工具不兼容，其他工具画的流程图用这个插件打开就乱了

.ignore

代码版本管理工具中，需要忽略的内容配置，可直接映射到目录，可以方便的查看和编写.gitignore文件

BackgroundImage

Idea背景设置，高逼格插件，可以设置Idea编辑窗口背景

Key Promoter X

Idea快捷键提示插件，鼠标操作Idea，会提示对应的快捷键

注： 以上插件都可以直接在Idea插件仓库中找到。
